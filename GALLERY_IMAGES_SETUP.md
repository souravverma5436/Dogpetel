### Troubleshooting

**Images not showing?**
1. Check if images are marked as "active" in admin panel
2. Verify image URLs are accessible (paste in browser)
3. Clear browser cache and refresh
4. Check browser console for errors

**Image URLs not working?**
- Use direct image URLs (ending in .jpg, .png, .jpeg)
- Imgur format: `https://i.imgur.com/XXXXX.jpg`
- Avoid shortened URLs or page links

**Gallery looks empty?**
- Run the automatic setup script (Method 1)
- Or manually add at least 3-4 images for best appearance

### Gallery Best Practices

1. **Image Quality**: Use high-resolution images (at least 800x800px)
2. **Variety**: Mix different activities, breeds, and settings
3. **Authenticity**: Real photos work better than stock images
4. **Consistency**: Maintain similar lighting and quality
5. **Regular Updates**: Add new images monthly to keep content fresh
